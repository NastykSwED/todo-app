# Todo App

This is a Todo App usign ReactJS and TailwindCSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/NastykSwED/todo-app
```

Go to the project directory

```bash
  cd todo-app
```

Install dependencies

```bash
 pnpm install
```

Start the server

```bash
  pnpm run dev
```
